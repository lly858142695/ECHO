# ECHO NEXT

ECHO NEXT 是面向本地曲库、桌面播放、HiFi 输出和长期维护的音乐播放器。

## 项目已转为闭源

由于一系列原因 或者可能是被骂烦了 作者认为开源没有任何好果子吃 所以：

ECHO NEXT 已正式转为闭源开发。将不再保留源码、项目架构、构建流程、内部文档、实现细节或开发路线的完整说明。

这次调整不代表项目停止维护，也不代表 ECHO NEXT 放弃长期更新。闭源对用户没有任何影响 我们会继续推进播放器本体、音频稳定性、曲库体验、界面体验、插件生态、跨平台适配和 Pro 能力；公开仓库后续主要用于项目说明、发布入口、反馈引导和开发者计划介绍。

出于维护成本、代码安全、商业授权、内部实验功能和长期开发节奏考虑，ECHO NEXT 的核心开发会在私有仓库和内部协作环境中继续进行。公开仓库中曾经出现过的架构说明、构建脚本、源码目录、技术细节和历史文档已经移除，请不要再以旧公开仓库内容作为当前项目结构依据。

## 我们仍然持续更新

ECHO NEXT 会继续维护和发布。后续更新重点包括：

- 提升本地曲库扫描、管理和大曲库浏览体验。
- 持续优化播放稳定性、输出设备兼容性和音频链路。
- 改进歌词、MV、封面、元数据和桌面集成体验。
- 继续完善插件、远程来源、Connect、Pro 权益和移动端相关能力。
- 修复用户反馈的问题，并围绕长期可维护性调整产品方向。

用户仍可通过官方发布渠道获取新版本。具体功能、发布时间和可用平台以正式发布说明为准。

## ECHO Pro

ECHO Pro 是给长期支持 ECHO NEXT 的用户准备的进阶计划。通过 ECHO Pro，你可以支持项目继续维护、更新和扩展，也可以获得后续开放的 Pro 能力、实验功能或相关权益。

ECHO Pro 支持入口：[https://afdian.com/a/echonext](https://afdian.com/a/echonext)

## ECHO Developer Plan

ECHO Developer Plan 是 ECHO NEXT 的开发者协作计划，面向希望更深入参与项目的人开放。加入后，你可以参与更早期的功能讨论、测试反馈、设计评审、文档维护和部分内部开发协作。

我们欢迎以下方向的协作者：

| 方向 | 可以参与的内容 |
| --- | --- |
| 前端 / 交互 | 播放器界面、歌词、MV、曲库、设置页、插件界面和体验打磨 |
| 后端 / 桌面 | Electron、Node、数据库、扫描、缓存、系统集成和诊断能力 |
| 原生 / 音频 | 音频输出、设备兼容、播放稳定性、平台适配和性能优化 |
| 美术 / 视觉 | UI 视觉、图标、插画、品牌素材、动效、宣传图和整体视觉一致性 |
| 测试 / 文档 | 问题复现、版本验证、用户教程、反馈整理和文档维护 |

Developer Plan 不只招代码贡献者。只要你愿意长期、认真、稳定地参与 ECHO NEXT 的建设，无论你擅长开发、设计、测试、文档、社区反馈还是产品体验，都可以申请。

申请时建议提供：

- GitHub ID。
- QQ 或其他可联系账号。
- 你熟悉的技术方向、项目经历或可展示作品。
- 如果申请设计 / 美术方向，请附作品集、截图、主页链接或说明你擅长的视觉范围。

参与过 ECHO / ECHO NEXT 贡献的用户可以优先申请。

Developer Plan 入口：[https://echonext.moe/zh/developer/](https://echonext.moe/zh/developer/)

要注意的是 潜水会被移除开发者计划~
